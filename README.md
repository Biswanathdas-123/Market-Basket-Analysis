# 🛒 Market Basket Analysis using Apriori Algorithm

## 📌 Project Overview

This project analyzes customer purchase transactions to identify products that are frequently purchased together using the Apriori Algorithm.

The insights from this analysis help retailers improve product placement, cross-selling strategies, inventory management, and promotional campaigns.

---

## 🎯 Objectives

- Analyze customer purchase behavior
- Discover frequently bought product combinations
- Generate association rules
- Improve retail decision making

---

## 🛠️ Tools & Technologies

- Apriori Algorithm
- Jupyter Notebook
