# Market Basket Analysis Project Report

## Purpose

The objective of this project is to analyze customer purchase patterns and identify products that are frequently bought together using the Apriori Algorithm.

## Business Problem

Retail stores need to understand customer buying behavior to improve product placement, cross-selling, inventory management, and promotional strategies.

## Methodology

1. Load transaction dataset
2. Clean the data
3. Convert transactions into basket format
4. Apply Apriori Algorithm
5. Generate Association Rules
6. Analyze Support, Confidence, and Lift

## Expected Results

- Frequently Bought Products
- Strong Association Rules
- Better Marketing Decisions
- Improved Product Placement

## Technologies Used

- Python
- Pandas
- Apriori Algorithm
- Mlxtend

## Dataset

Groceries Dataset

## Conclusion

Market Basket Analysis helps retailers understand customer purchasing behavior and make data-driven business decisions.
